# User Manual

## Roles

- `Admin`: Manage users, system activity, subscriptions, logs, and platform settings.
- `Business Analyst`: Manage customers, train models, export reports, and analyze dashboards.
- `Standard User`: View dashboards, reports, and profile settings.

## Core Workflow

1. Register or sign in.
2. Import or seed customer data.
3. Train the segmentation model.
4. View dashboard metrics and AI recommendations.
5. Export reports for stakeholders.

- `admin@segmify.ai / Admin@123`
- `analyst@segmify.ai / Analyst@123`
